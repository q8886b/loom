# Code of Conduct

We want Loom to be a calm, rigorous place to build local-first agent tooling.

Participants are expected to:

- Be respectful and concrete in technical discussion.
- Critique ideas and changes, not people.
- Avoid sharing private, copyrighted, or sensitive material in issues and pull
  requests.
- Assume good intent while still asking for evidence when correctness matters.

Maintainers may remove comments, issues, or contributions that are abusive,
harassing, spammy, or unsafe for a public project.
